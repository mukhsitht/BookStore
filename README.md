<div class="markdown prose w-full break-words dark:prose-invert light"><h1>Bookstore Management Application - README</h1><h2>Overview</h2><p>This Bookstore Management Application allows users to manage a catalog of books, performing CRUD (Create, Read, Update, Delete) operations on books. Users can manage book details like title, author, price, and publication date. The application is built with a modern and responsive frontend using <strong>Angular 15</strong> and <strong>TailwindCSS</strong>, and a backend API built with <strong>.NET Core</strong> to handle all data processing and persistence with a database. The design emphasizes clean architecture, proper error handling, scalability, and ease of maintenance.</p><h2>Features</h2><h3>1. Book Listing Page</h3><ul><li>Displays a list of all books stored in the system.</li><li>Allows sorting and filtering by author, price range, and publication date.</li><li>Responsive table layout for ease of browsing on different devices.</li></ul><h3>2. Book Details Page</h3><ul><li>Shows detailed information for a selected book (title, author, price, and publication date).</li><li>Provides options to edit or delete the selected book.</li></ul><h3>3. Add/Edit Book Form</h3><ul><li>A form to add new books or update existing books.</li><li>Client-side form validation ensures required fields like title, author, and positive price values are filled out correctly.</li></ul><h3>4. User Interface (UI)</h3><ul><li>Built with <strong>TailwindCSS</strong> for responsive, mobile-friendly design.</li><li>Clean, minimalistic design focusing on usability and accessibility.</li><li>Consistent layout across all pages (listing, details, and forms).</li></ul><h3>5. Error Handling &amp; User Feedback</h3><ul><li>Meaningful error messages displayed during form validation failures (e.g., empty required fields or invalid data types).</li><li>Success messages shown after successful book additions or updates.</li></ul><h2>Technology Stack</h2><h3>Frontend</h3><ul><li><strong>Angular 15</strong>: A framework for building dynamic single-page applications.</li><li><strong>TailwindCSS</strong>: A utility-first CSS framework for building responsive designs quickly.</li></ul><h3>Backend</h3><ul><li><strong>.NET Core</strong>: Handles API requests and data persistence.</li><li><strong>Entity Framework Core</strong>: ORM for database operations.</li></ul><h3>Database</h3><ul><li><strong>SQL Server</strong> or any relational database of choice (configurable).</li></ul><h2>Setup Instructions</h2><h3>Prerequisites</h3><ul><li><strong>Node.js</strong> (v14 or higher)</li><li><strong>Angular CLI</strong> (v15)</li><li><strong>.NET Core
